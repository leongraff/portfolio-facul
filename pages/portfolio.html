Aplicativo mobile para cafeteria ( imagens do projeto+ )
A aplicação, desenvolvida com Nuxt (Vue) e MongoDB, foi criada para atender os clientes de uma cafeteria em Itajaí (V.ON), oferecendo diversas funcionalidades para otimizar a experiência do usuário.
Entre os principais recursos, destaco:
  • Cadastro e login de usuários;
  • Criação de pedidos a partir de um carrinho de compras;
  • Sistema de cashback e créditos especiais na plataforma;
  • Cadastro e alteração de múltiplos endereços de entrega;
  • Histórico de cashback, créditos e pedidos; Integração com diferentes formas de pagamento via Asaas (cartão e Pix);
  • Integração completa com o sistema de PDV desenvolvido pela Ws Sistec.
